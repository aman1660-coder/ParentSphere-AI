## рҹ“Ң About ParentSphere

**ParentSphere** is a full-stack parenting support and child development platform designed to solve a real-world problem: parents often need to use multiple disconnected applications for counselling, child development tracking, educational resources, parenting guidance, and community interaction.

ParentSphere brings these services together into a single, modern, and user-friendly platform.

The application allows parents to:

- рҹӨ– Receive AI-powered parenting guidance
- рҹ‘¶ Create and manage child profiles
- рҹ“Ҳ Track child growth and development
- рҹ‘©вҖҚвҡ•пёҸ Discover professional counsellors
- рҹ“… Book and manage counselling appointments
- рҹ’і Track counselling payments
- рҹ“ҡ Access parenting books and educational resources
- рҹ“° Read practical parenting articles
- рҹ’¬ Participate in community discussions
- рҹҺҘ Watch parenting courses, webinars, and child development videos
- рҹ“Ҡ Access a personalized parent dashboard

The application follows a modern **MERN stack architecture** and is deployed using independent frontend, backend, and cloud database services.

# вҡҷпёҸ Getting Started

Follow the instructions below to run ParentSphere locally.

## 1пёҸвғЈ Clone the Repository

```bash
git clone YOUR_GITHUB_REPOSITORY_URL
```

Navigate to the project directory:

```bash
cd Parentsphere
```

---

## 2пёҸвғЈ Install Project Dependencies

Install root dependencies:

```bash
npm install
```

Install frontend dependencies:

```bash
cd frontend
npm install
```

Install backend dependencies:

```bash
cd ../server
npm install
```

---

## 3пёҸвғЈ Configure Environment Variables

Create a `.env` file inside the `server` directory.

```env
NODE_ENV=development

PORT=5000

CLIENT_URL=http://localhost:5173

MONGODB_URI=your_mongodb_atlas_connection_string

JWT_ACCESS_SECRET=your_secure_access_token_secret

JWT_REFRESH_SECRET=your_secure_refresh_token_secret

ACCESS_TOKEN_EXPIRES=15m

REFRESH_TOKEN_EXPIRES=7d

OPENAI_API_KEY=your_api_key_if_required

RAZORPAY_KEY_ID=your_key_if_required

RAZORPAY_KEY_SECRET=your_secret_if_required
```

> вҡ пёҸ Never commit `.env` files, MongoDB credentials, JWT secrets, API keys, or payment credentials to GitHub.

---

## 4пёҸвғЈ Seed the Database

ParentSphere includes a database seeding script that can populate the database with initial application content.

Navigate to the server directory:

```bash
cd server
```

Run:

```bash
node seed/seed.js
```

Expected output:

```text
MongoDB connected successfully.

Deleting existing counsellors, books, articles and videos...

Adding new data...

========================================
DATABASE SEEDED SUCCESSFULLY
========================================

Counsellors: 6
Books: 4
Articles: 4
Videos: 3

========================================
```

---

## 5пёҸвғЈ Start the Backend Server

From the `server` directory:

```bash
npm run dev
```

The development backend will run on:

```text
http://localhost:5000
```

---

## 6пёҸвғЈ Start the Frontend

Open another terminal and navigate to the frontend directory:

```bash

```

```bash

```

```text
http://localhost:5173
```
